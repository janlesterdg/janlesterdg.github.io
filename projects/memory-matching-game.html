Memory Matching Game

A matching game to test the player's memory by pairing up the cards with the same pictures.

VISIT
CODE

Technologies and Frameworks:

HTML5, CSS3, JavaScript

Features:

  • Timer

    • When a player starts a game, a displayed timer should also start. Once the player wins the game, the timer stops.

  • No. of Moves

    • The game displays the current number of moves a user has made.

  • Rating

    • The game shows a rating that reflects the player’s performance based on the number of moves made.

  • Shuffle

    • Cards are to be shuffled on load or restart.

  • Modal

    • A congratulation modal appears when the player wins while showing a button to play again. The modal will show the details of the number of moves, timer, and the rating of the player.

Demo:

The goal of the game is to match all the 16 cards (8 pairs) to win the game. The game cards are flashed all at once when the page reloads or restarts. When a player starts a game, a displayed timer should also start. The number of moves increments after the player has selected two cards. A restart button will allow the player to reset the game board, the timer, the moves, and the player rating. Once the player wins the game, the timer stops.


A congratulation modal should appear when the player wins. The modal will show the details of the number of moves, timer, and the rating of the player. The modal has functions for closing the modal or resetting the game by clicking the "Play Again" button at the bottom. The “Play Again” button shuffles the cards, removes all the extra classes, removes images, resets moves to 0, resets the timer, and restores the rating to 5 stars with an opacity of 1.
