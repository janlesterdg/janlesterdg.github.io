Scoreboard

A scoreboard application that allows users to keep track of players' scores.

VISIT
CODE

Technologies and Frameworks:

HTML5, CSS3, JavaScript, React

Components:

  • App (orange)

    • The App component contains the player objects and the props of Header and Player components. This component is in charge of rendering all of the components to the root container.

  • Header (green)

    • The Header component displays the title and the total number of players.

  • Player (blue)

    • The Player component displays a list of players with its corresponding score and allows the user to remove a player.

  • Counter (blue)

    • The Counter component allows the user to change the state of the player's score by incrementing and decrementing it with an onClick event.

Demo:

The scoreboard has 4 predefined players. Each player has their score counter. The user can increment and decrement the score by 1 point by clicking the "+" or "-" button. Players can also be removed by clicking the "x" button and the number of players on the header will be updated.
