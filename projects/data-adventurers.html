Data Adventurers

A system that allows a student organization to post announcements, activities, etc.

CODE

Technologies and Frameworks:

HTML5, JavaScript, Bootstrap, jQuery, PHP, MySQL

Modules:

  • Super Admin (Organization Advisers)

    • - user management (enable/disable accounts)

    - update page information

    - accept/reject membership registration

    - post announcements/activities/memo

    - create/hide polls

    - can change logo and update page information

    - can set-up the start of school year and registration date


  • Admin (Organization Officers)

    • - update page information

    - accept/reject membership registration

    - post announcements/activities/memo

    - create/hide polls

    - can change logo and update page information


  • Clients (Member)

    • - account registration

    - view page info/ annoucements (notified via email for any new posts/activities)

    - participate in polls

    - sign up school activities posted by the admin

Demo:

This is the landing page of Data Adventurers where existing members can log in and new members can sign up an account.


The super admin can enable and disable accounts, update page information and logo, accept/reject membership registrations that are pending, post announcements/activities/memos, create/hide polls, edit their profile, and set-up the start of the school year and registration date.


The admin can enable and disable accounts, update page information and logo, accept/reject membership registrations that are pending, post announcements/activities/memos, create/hide polls, and edit their profile.


The client can create an account, view page information, memo, activities, announcements (notified via email for any new posts/activities), participate in polls, sign up school activities posted by the admin, and edit their profile.
